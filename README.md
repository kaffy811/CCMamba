# CCMamba: Complementary Masked Siamese Mamba Network

本仓库实现了基于 **Sigma**  架构改进的多模态语义分割算法。针对原框架在极端环境下的鲁棒性不足，我们引入了 **互补随机掩码 (CRM)**、**教师-学生自蒸馏** 以及 **置信度加权融合** 机制。

---

## ## 核心技术范式

本框架在保留 **Mamba** 线性复杂度与全局建模优势的基础上 ，从以下三个维度对原算法进行了演进：

### ### 1. 互补随机掩码 (Complementary Random Masking, CRM)

为了打破模型对单一模态的过度依赖（Modality Bias），我们在数据预处理阶段引入了 CRM 策略：

* **原理**：生成一个随机的二值化掩码 $M$，将其施加于 RGB 分支，同时将补集掩码 $(1-M)$ 施加于 X-Modal（红外/深度）分支。
* **优势**：强制网络在任一模态局部信息缺失时，必须通过另一模态的对应区域进行特征补偿，显著提升了传感器失效场景下的鲁棒性。

### ### 2. 教师-学生自蒸馏 (Self-Distillation Stream)

针对掩码导致的特征分布偏移，我们构建了双路训练范式：

* **Teacher Stream**：输入完整图像，在无梯度状态下生成“干净”的预测结果（Clean Logits）作为监督信号。
* **Student Stream**：输入掩码图像，通过 **KL 散度** 学习教师路径的高维特征分布。
* **损失函数**：$L_{total} = L_{seg} + \alpha \cdot L_{distill}$，引导学生网络重构出缺失的语义细节。

### ### 3. 置信度加权门控融合 (Confidence-Gated Fusion)

原 Sigma 框架的矩阵 $C$ 交换机制假设特征全局可靠 。我们在其基础上增加了显式门控：

* **逻辑**：在特征进入 **CroMB** 融合块前，通过全局平均池化（GAP）与 Sigmoid 激活生成空间置信度权重 $\alpha$。
* **效果**：模型能自动识别并抑制被掩码遮挡或低质量的特征区域，确保多模态融合过程聚焦于高置信度的有效像素。

## ## 核心数据集与评估指标

* 
**数据集**：支持 **MFNet** (RGB-T) 、**PST900** (RGB-T)  等。


* 
**指标**：采用 **mIoU** (Mean Intersection over Union) 衡量分割精度 。



---

## ## 复现与挑战

* 
**优势**：相比传统 CNN 范式，本框架通过 Mamba 的 **SS2D** 扫描实现了线性级全局感受野 ，且通过自蒸馏机制有效缓解了多模态数据的不对齐问题。


* **核心挑战**：
1. **算子依赖**：`Selective Scan` 算子高度依赖 CUDA 加速，在非 NVIDIA 环境（如 MacBook M4）下需针对性适配。
2. **训练显存**：双路流训练对显存有一定要求，本代码已通过 `torch.no_grad()` 优化教师路径，适配 **RTX 3090** 等单卡环境。
